# 对焦景深虚化工具（本地初版）

一个零依赖、浏览器可用的小工具：上传图片后，手动设置对焦点，叠加透视参考，生成景深虚化并下载结果图。

## 功能

- 上传图片并在 Canvas 中预览
- 上传后自动估算透视点（可手动 Shift+点击改透视点）
- 点击设置对焦点（绿色点）
- Shift + 点击设置透视点/消失点（蓝色点）
- 可调参数：虚化强度（1-5）、椭圆对焦半径X/Y、过渡宽度、透视影响、景深曲线偏移
- 可选高光特效：背景高光 bokeh 开关、圆形/六边形光斑、星芒强度
- 预览自动适配视窗显示
- 下载保持原图分辨率（不缩放）
- 生成效果图并下载 PNG

## 本地使用

### 方式 A：直接打开

直接双击 `index.html`（部分浏览器在本地文件模式下也可用）。

### 方式 B：本地静态服务（推荐）

在项目目录运行：

```bash
python3 -m http.server 5173
```

然后访问：

`http://localhost:5173`

## 部署与分享（GitHub Pages）

本项目已内置自动部署文件：`.github/workflows/deploy-pages.yml`。

按下面 4 步即可上线：

1. 将当前目录推送到 GitHub 仓库（分支名 `main`）。
2. 在仓库 `Settings -> Pages` 中，将 `Source` 设为 `GitHub Actions`。
3. 推送代码后等待 Actions 跑完（约 1-2 分钟）。
4. 访问你的站点：`https://<你的GitHub用户名>.github.io/<仓库名>/`

如果你的仓库是私有仓库，请确认账号套餐支持 Pages，或者改为公开仓库。

## 说明

- 当前是初版，透视分析为轻量估算，不是严格几何重建。
- 大图处理会更耗时（例如 3840x3840），建议先调好参数再生成最终图。
